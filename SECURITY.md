# Security Policy

## Supported Versions

The latest `main` branch is supported.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory or by emailing the repository owner.

Do not include private Photos library data in public issues. Dry-run output may contain filenames, dates, and local Photos identifiers.

## Privacy Notes

iCloud Screenshot Sweeper runs locally. It does not upload files, filenames, identifiers, or library metadata to a remote service.
